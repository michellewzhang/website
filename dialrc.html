dialog schema builder

∙ full stack development of a drag-and-drop based application to build & test dialog schema

∙ sept 2020 - present

overview

I'm currently working for the DialRC Group (part of CMU Language Technologies Institute) to develop a multifunctional application with React that allows users to build flowchart-based representations of schema using drag-and-drop components and test their implementations in real time with a chatbox UI.

To learn more about this ongoing project, please contact me!
